# Niyam ‚Äî Visual Identity v2 ("Sunrise Sans") Design

**Date:** 2026-06-11 ¬∑ Sub-project 8
**Status:** Founder-approved via interactive mockup session (this spec transcribes those decisions; the mockups are the contract).

## 1. Purpose

Restyle every screen from the flat "text on cream" v1 into the founder-approved v2 language: warm sunrise-gradient canvases, floating cards, a modern geometric sans (Outfit), purposeful micro-motion. No content, navigation, or logic changes.

**In plain English:** same app, same screens, same words ‚Äî but it stops looking like a form and starts feeling like a warm, modern, alive product.
